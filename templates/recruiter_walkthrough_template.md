# Recruiter Walkthrough Template

## 1. Setup

- I built a Python-based peer benchmarking and valuation project using only free data sources.
- The project pulls market and macro data, calculates relative performance and valuation metrics, and adds a DCF layer.

## 2. What the project outputs

- Chart pack with performance, valuation, correlation, scorecard, and DCF visuals
- Excel workbook for table review
- HTML report for browser-based presentation
- Streamlit dashboard for interactive live walkthroughs

## 3. How I would tell the story

- Start with why the peer set matters strategically
- Compare performance and quality across names
- Explain which stocks screen as value versus quality
- Show how macro conditions affect valuation and capital markets sentiment
- Use the DCF sensitivity table to explain how judgment changes implied value
