# Sample Outputs

This folder is meant for GitHub publication.

It contains a curated subset of generated deliverables so recruiters can quickly review the project without opening Python files or rerunning the workflow.

Suggested contents:

- `dashboard_report.html`
- `executive_summary.md`
- `recruiter_case_study.md`
- selected charts in `charts/`

Working outputs generated by each run remain in `outputs/` and are intentionally ignored by Git.
